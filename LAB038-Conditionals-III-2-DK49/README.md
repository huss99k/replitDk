Given 2 int values, return true if they are both in the range 30..40 inclusive,
or they are both in the range 40..50 inclusive.

 Examples:
  in3050(30, 31) → true
  in3050(30, 41) → false
  in3050(40, 50) → true
  in3050(40, 51) → false
  in3050(39, 50) → false
  in3050(49, 48) → true
  in3050(50, 51) → false
  in3050(35, 45) → false
