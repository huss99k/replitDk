```
String str
```

```
int start
```

```
int end
```

Print out the following string:

The substring of (str) from (start) to (end) is (substring from start to end, inclusive)

```
In: lolwut
```

```
Start: 2
```

```
End: 4 
```

```
The substring of lolwut from 2 to 4 inclusive is lwu
```
